🎬 Movie Recommendation System

A Machine Learning based Movie Recommendation System built with Python and Streamlit that suggests similar movies based on user selection.
The project provides a simple and interactive UI where users can discover movies instantly.

⚙️ Installation
1. Clone the Repository
git clone https://github.com/keshaviit/Movie_Recommendation_System.git
2. Navigate to the Project Folder
cd Movie_Recommendation_System
3. Install Dependencies
pip install -r requirements.txt
4. Run the Project
streamlit run app.py
🧠 Recommendation System Logic

This project uses content-based filtering to recommend movies.
The system compares movie features and similarity scores to suggest related movies to users.
